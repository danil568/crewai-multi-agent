# CrewAI Multi-Agent System

Мультиагентная система на CrewAI — два агента работают как команда.

## Как работает
1. Агент-Исследователь изучает тему и собирает факты
2. Агент-Писатель берёт факты и пишет статью

## Технологии
- Python 3.11
- CrewAI
- Claude Haiku через OpenRouter

## Запуск
1. Создай .env с ключом OPENAI_API_KEY
2. pip install crewai
3. python main.py
